# Images de Lémurien pour Lummy

Ce dossier contient les images de la mascotte Lummy (lémurien).

## Images nécessaires

Veuillez télécharger et placer les images suivantes dans ce dossier :

1. **lemur-neutral.jpg** - Lémurien en position neutre/repos (400x400px recommandé)
2. **lemur-happy.jpg** - Lémurien souriant/joyeux (400x400px recommandé)
3. **lemur-thinking.jpg** - Lémurien pensif/curieux (400x400px recommandé)
4. **lemur-excited.jpg** - Lémurien excité/enthousiaste (400x400px recommandé)

- Taille recommandée: 400x400px minimum
